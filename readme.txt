Instruction:
  1. Compile, locate to the folder and run 'make', the Makefile will generate two executable files, namely 'server' and 'client'
  2. The usage of server is ./server [portnumber], run server on a computer first.
  3. The usage of client is ./client [IP_Address][PORT_Nunber], run this client executable on another computer, and it will connect to the server.
  4. Communicate, the program uses standard input for communication. Typing text in command line on either side will send the text to the other.
